安装

通过 npm、Composer 或 Meteor 安装 Bootstrap 的源 Sass 和 JavaScript 文件。

使用包管理器安装的 Bootstrap 不包含文档和全部的构建脚本。 你也可以使用 我们的 npm 模版仓库 来快速生成一个 Bootstrap 项目。
